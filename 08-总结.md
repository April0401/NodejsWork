# 总结

- Node.js是什么
    + JavaScript运行时
    + 既不是语言，也不是框架，它是一个平台
- Node.js中的JavaScript
    + 没有BOM、DOM
    + EcmaScript基本的JavaScript语言部分
      * 变量
      * 方法
      * 数据类型
      * 内置对象
      * Array
      * Object
      * Date
      * Math
- Node.js为JavaScript提供了一些服务器级别的API
    + 文件操作能力
    + http服务能力
- Node.js模块系统
    + 没有全局作用域的概念
    + 通过require加载多个 JavaScript脚本文件
    + 模块提供了exports接口对象
